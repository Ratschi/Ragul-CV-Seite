Minetest Server mit Terraform

Aufgabenstellung

Mein Ziel ist es einen Minetest Server zu erstellen auf welchem ich mit Freunden spielen kann. Minetest ist die Open-Source Version von Minecraft. Der Minetest Server soll auf einer Ubuntu VM in der Azure Cloud gehostet werden. Damit es ein wenig spannender wird, wollte ich das ganze mit Hilfe von Terraform machen. Ich nutze den Azure Tenant meiner Firma.

Umgebung und Netzwerkplan

Bevor ich mit dem Erstellen der Ressourcen starte, habe ich einen Netzwerkplan erstellt. Dies habe ich gemacht um genau zu planen welche Ressourcen ich dabei haben möchte. Der Netzwerkplan zeigt alle Ressourcen an, die ich für dieses Projekt benötige.

Terraform Deployment

Wer mich kennt, weiss das ich Dinge gerne simpel halte, daher nutze ich nur ein "Main.tf". Das Main.tf ist das Hauptkonfiguration File von Terraform und beinhaltet die Ressourcen die ich erstellen möchte. Folgende Ressourcen müssen erstellt werden: virtuelle Maschine (Ubuntu), virtuelles Netz mit einem Subnet, öffentliche IP-Adresse und eine Netzwerkkarte. Alle Ressourcen werden in einer vorhandenen Ressourcengruppe erstellt. Wer möchte, kann meinen gerne kopieren und für sich selber nutzen.

Es fehlt noch eine wichtige Sache, der Minetest Server. Anstatt die Installation manuell zu machen, nutze ich ein cloud-init-File.
